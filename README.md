# NTNU_Moodle_ColorTheme
台師大 Moodle 網頁 個性顏色主題

### 使用方法
```
下載對應顏色的資料夾 > 打開 browser > 擴充功能 > 載入未封裝項目 > 選擇整個資料夾
安裝好後只要每次打開Moodle，就會改變網頁樣式
```

